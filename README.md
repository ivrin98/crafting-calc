This is an offline portable Lineage 2 C5 recipe calculator.

How to open:

1. Go to releases
2. download the Source code.zip
3. Unzip it anywhere you want
4. Open the unzipped folder
5. Launch Recipe Crafter 1.0.0.exe

Featues:
1. Dark mode (most important)
2. Have multiple recipe material lists at once

How does it work:
Pick a recipe (or multiple). Material graph is generateed in the bottom.
Craftable materials - everything that can be crafted, including craftable mateirals that are a part of the recipe
Raw Materials - everything that is needed but cannot be crafted

The calculator sums everything up and sorts it alphabetically so it's easier to find. Everything that's already collected disappears from the list.
For example, if you need 8 crafted leather but you already have it, both crafted leather and whatever's needed to craft it wil disappear form the list.
